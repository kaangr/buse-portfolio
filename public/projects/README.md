# Project assets

Drop final files into each project folder. Paths match `src/data/projects.ts`.

```
projects/
  helia-restaurant/
    cover.svg          → replace with cover.jpg / cover.webp (cover can stay SVG)
    renders/           → PNG only: render-01.png, render-02.png, …
    plans/             → floor plans (plan-01.pdf, …)
    docs/project.pdf   → presentation PDF
  tekel-tobacco/
  organic-market/
  portfolio-website/
    docs/case-study.pdf
```

**Renders:** Use `.png` files under each project's `renders/` folder. They appear in the project detail panel and open full-size when clicked.
